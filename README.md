Source for an analysis project hosted on Kaggle.

You might be here for the cs.js. Install node then run `npm i`. Make sure that the directory strings in `cs.js` point to the correct files and then run `node cs.js`.

The data files are not included due to size. You can find them on [Kaggle](https://www.kaggle.com/datasets/arashnic/fitbit) and on [Zenodo](https://zenodo.org/record/53894).
